# Module 12 Assignment: Business Analytics Fundamentals and Applications

## Project Title
**GreenGrocer Data Analysis**

## Description
This project performs an end-to-end data analysis for "GreenGrocer," a retail chain with multiple locations across Florida. The analysis utilizes Python to process simulated sales, customer, and operational data to derive actionable business insights. The script covers the full spectrum of business analytics:
* **Descriptive Analytics:** Summarizing historical sales and profit performance.
* **Diagnostic Analytics:** Identifying correlations between store size, staffing, and revenue.
* **Predictive Analytics:** Using linear regression to forecast sales based on physical store metrics.
* **Prescriptive Analytics:** Developing strategic recommendations for margin optimization and operational efficiency.

## Files Included
* `greengrocer_analysis.py`: The main Python script containing data generation, analysis functions, and visualization logic.
* `README.md`: Documentation for the repository.

## What I Practiced
In this assignment, I applied several core data science and business intelligence skills:
* **Data Wrangling:** Using `pandas` to group, merge, and resample time-series data.
* **Data Visualization:** Creating bar charts, pie charts, and heatmaps with `matplotlib` to communicate complex data distributions.
* **Statistical Modeling:** Implementing linear regression using `scipy.stats` to predict future performance.
* **Executive Communication:** Translating technical findings into a concise executive summary with specific, data-driven business recommendations.
